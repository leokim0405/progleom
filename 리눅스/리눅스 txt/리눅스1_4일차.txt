:! 명령어
- vi를 잠시 중단하고 입력한 리눅스 명령어를 실행
ex) :! ls -l /home

:.! 명령어
- 명령어 실행 결과를 현재 편집하고 있는 파일에 삽입
- 현재 커서가 있는 라인을 지우고 명령어의 결과를 삽입하기 때문에
  신중하게 사용해야 합니다

:q
- vi 에디터 종료

:w
- 파일 저장

:wq
- 저장하고 vi 에디터 종료

:w [경로]
- 다른 이름으로 저장
ex):w /home/samadal/login

:q!
- 저장하지 않고 vi 에디터 종료

:e 파일명
- 지정한 파일 불러오기
ex) :e /backup/inittab

:e! 파일명
- 저장하지 않고 파일 불러오기
ex) :e! /backup/login.defs

:enew
- 새로운 파일 불러오기

:w >> 파일
- 현재 열려있는 파일의 내용을 지정한 파일 마지막에 삽입

:r 파일
- 현재 커서 아래에 지정한 파일 내용을 삽입

:r! 명령어
- 현재 커서 아래에 지정한 명령어의 결과를 삽입

사용자 계정(User)
- 사용자는 OS를 사용하기 위해서 반드시 필요한 하나의 객체입니다
- 우리는 사용자를 통해서 OS에 접속해서 OS를 관리 및 운용합니다
- 사용자는 관리자 계정과 일반 사용자 계정으로 분류합니다

/etc/passwd
- passwd 파일은 리눅스에서 사용하는 모든 계정의 정보를 가지고 있는 파일입니다
- 실제 계정을 사용할 때 passwd안에 들어있는 정보를 가지고 계정에 로그인을 합니다
- passwd에 잘못된 계정 정보가 들어있다면 그 계정을 사용할 수 없습니다

passwd 파일 구조

[root@localhost ~]# grep samadal /etc/passwd
samadal:x:1000:1000:samadal:/home/samadal:/bin/bash
[root@localhost ~]#

samadal = 사용자 이름
x = 패스워드
1000 : 1000 = UID(User ID) : GID (Group ID)
samadal = 주석(코멘트)
/home/samadal = 사용자 계정의 홈 디렉터리 경로
홈 디렉터리 = 숨겨진 사용자 설정 파일(skel 파일)을 가지고 있는 디렉터리를
   	    사용자 홈 디렉터리 라고 합니다
/bin/bash = 사용자가 기본적으로 사용하는 셸(명령어 처리기)

[사용형식]
useradd [옵션1] [옵션값1] [계정명]

-d = 사용자의 홈 디렉터리 생성 위치를 설정하는 옵션
# mkdir /itbank
