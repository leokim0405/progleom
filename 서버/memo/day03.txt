도메인 가입의 종류
 - 도메인의 멤버로 가입
    > 컴퓨터를 도메인의 일반 멤버로 사용

	컴퓨터 속성 -> 작업 그룹 변경

	  * 도메인을 찾아갈 수 있게 IP설정 -> DNS 지정

 - 도메인의 보조 DC로 가입
    > 컴퓨터를 도메인의 추가 DC로 사용

	멤버로 가입 -> DC로 승격 -> 기존 도메인에 도메인 컨트롤러 추가

 - 도메인의 서브 도메인을 생성
    > 컴퓨터를 도메인의 하위 도메인을 구성하는 DC로 사용

	DC로 승격 -> 기존 포리스트에 새 도메인 추가 -> 자식 도메인

	  * 상위 도메인의 가입 권한이 있는 계정(관리자 등) ID/PW 필요
	     + 상위 도메인을 찾아갈 수 있게 IP설정 -> DNS 지정

  * 포리스트에 다른 트리 도메인 추가
     > 기존 트리가 아닌 새로운 도메인 트리 생성

	DC로 승격 -> 기존 포리스트에 새 도메인 추가 -> 트리 도메인

	  * 포리스트의 가입 권한이 있는 계정(관리자 등) IP/PW 필요
	     + 포리스트의 루트 도메인을 찾아갈 수 있게 IP설정 -> DNS 지정

도메인 관리 도구
 - 도메인에 멤버로 가입한 컴퓨터에는 기본적으로 도메인 관리 도구가 설치가 되어있지 않다
    > 도메인 관리 도구를 설치해야 사용 가능

	서버관리자 -> 역할 및 기능 추가 -> 역할 X -> 기능 -> 원격 서버 관리 도구 -> 역할 관리 도구
	-> AD DS 및 AD LDS 도구 -> AD DS 도구 체크 -> 기능 추가 -> 설치 진행

    > 이 후 부터는 멤버 컴퓨터에서 도메인 관리가 가능
