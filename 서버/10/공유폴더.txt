공유 폴더
  - 같은 네트워크에서 파일을 배포하기 위해 사용하는 폴더
     > 요즘은 USB 등의 휴대용 저장 매체를 사용하거나 이메일로 배포

  - 폴더를 공유하고 그 폴더 안에 공유할 파일을 넣어서 사용
    > 개별 파일은 공유될 수 없다

  - 기본적으로 폴더를 공유한 사용자는 해당 공유 폴더에 대해 모든 권한을 획득

  - 공유 폴더 목록 확인 방법
    1. 아이콘 변화
    2. cmd > net share
    3. 컴퓨터 관리 -> 공유

compmgmt.msc -> 컴퓨터 관리

  - 공유 이름 뒤에 $를 붙여서 숨김 공유를 할 수 있다.
    > 숨김 공유 폴더에 접근하기 위해 \\IP\공유이름$ 형식으로 접근

  - 관리 공유 폴더
     > 공유 목록에 기본적으로 존재하는 공유 폴더
        특수한 목적으로 사용

    > C$, D$ 와 같이 드라이브 문자에 $가 붙은 경우에는 해당 드라이브를 관리하기 위하여 생성
      장치(파티션)가 인식 되면 자동으로 생성 (CD-ROM, USB 는 생성되지 않음)
    
    > Admin$ 는 운영체제가 설치되어 있는 C:\windows 폴더를 관리
      (환경변수 : %systemroot%)
 
    > Print$는 프린터를 공유했을 때 다른 컴퓨터가 프린터를 정상적으로 사용할 수 있도록 드라이버를 넣어놓는 폴더

    > IPC$ 는 서버간의 통신에 사용

  - 공유 권한
    > 읽기		: 파일 내용과 속성을 볼 수 있다
		 하위 폴더를 열 수 있다
		 공유 폴더 내부의 프로그램을 실행
      
      읽기/쓰기	: 읽기 항목의 모든 권한 포함
		  파일과 하위 폴더를 생성 할 수 있다
		  파일의 내용을 변경 할 수 있다 (공유권한은 기본적으로 상속)
		  파일과 하위 폴더를 삭제 할 수 있다

      소유자	: 읽기/쓰기의 모든 권한 포함
		  공유 폴더의 보안 설정까지 변경 가능

      * 공유 권한은 NTFS 권한과 AND 연산 된다. (둘다 허용일때만 허용)
